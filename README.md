## Getting Started

To run this project locally, ensure you have Node.js and npm installed. You can use [nvm](https://github.com/nvm-sh/nvm#installing-and-updating) to manage Node.js versions.

### Steps:

```sh
# 1. Clone the repository
git clone <YOUR_GIT_URL>

# 2. Navigate to the project directory
cd kenya-bin-buddy-main

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

## Deployment

To deploy this project, build the production assets and serve them using your preferred hosting solution:

```sh
# Build for production
npm run build

# Preview the production build locally
npm run preview
```

You can then deploy the contents of the `dist` folder to your static hosting provider (e.g., Netlify, Vercel, GitHub Pages, etc.).
